Memory Architecture
• Short-term: In-memory ring buffer (50 entries), fast access for recent context
• Long-term: ChromaDB vector database with semantic search (cosine similarity)
• Retrieval: Automatic RAG — relevant memories injected into every agent prompt
• Model: all-MiniLM-L6-v2 for embeddings
